# Руководство пользователя TravelPi

## Быстрый старт

### Подключение к роутеру
1. Включите Raspberry Pi с установленным TravelPi
2. Найдите Wi-Fi сеть **TravelPi**
3. Подключитесь с паролем: `TravelPi2024`
4. Откройте браузер и перейдите на: http://192.168.4.1 или http://travelpi.local

### Первый вход
- **Логин**: admin
- **Пароль**: admin123

⚠️ **Обязательно смените пароль после первого входа!**

## Основные функции

### 1. Главная страница
- Мониторинг CPU, памяти и температуры
- Статус подключения к интернету
- Список подключенных устройств

### 2. Управление сетью

#### Режимы подключения к интернету:
- **USB Wi-Fi** - подключение через внешний Wi-Fi адаптер
- **USB LAN** - подключение через Ethernet адаптер
- **Авто** - автоматический выбор (приоритет LAN)

#### Подключение к Wi-Fi:
1. Нажмите "Сканировать"
2. Выберите нужную сеть
3. Введите пароль
4. Нажмите "Подключить"

### 3. Управление устройствами

#### Блокировка устройств:
1. Перейдите в раздел "Устройства"
2. Найдите нужное устройство
3. Нажмите "Заблокировать"

#### Именование устройств:
- Кликните на поле с именем устройства
- Введите новое имя
- Изменения сохраняются автоматически

### 4. Настройки

#### Изменение точки доступа:
1. Введите новое имя сети (SSID)
2. Введите новый пароль (минимум 8 символов)
3. Нажмите "Сохранить"
4. Переподключитесь к новой сети

#### Блокировка рекламы:
- Включите/выключите переключатель
- Для детальной настройки откройте панель Pi-hole

#### Смена пароля администратора:
1. Введите текущий пароль
2. Введите новый пароль
3. Подтвердите новый пароль
4. Нажмите "Изменить пароль"

## Типичные сценарии использования

### Сценарий 1: В отеле с Wi-Fi
1. Подключите USB Wi-Fi адаптер
2. Выберите режим "USB Wi-Fi"
3. Отсканируйте сети
4. Подключитесь к Wi-Fi отеля
5. Все ваши устройства теперь подключены через TravelPi

### Сценарий 2: В месте с Ethernet
1. Подключите USB LAN адаптер к сети
2. Выберите режим "USB LAN" или "Авто"
3. Интернет появится автоматически

### Сценарий 3: Защита детских устройств
1. Включите блокировку рекламы
2. Настройте Pi-hole для фильтрации контента
3. Заблокируйте нежелательные устройства

## Технические характеристики

### Сеть точки доступа:
- **IP адрес роутера**: 192.168.4.1
- **Диапазон DHCP**: 192.168.4.2 - 192.168.4.20
- **DNS**: Pi-hole + Google DNS (8.8.8.8)

### Поддерживаемые устройства:
- Смартфоны (iOS, Android)
- Ноутбуки (Windows, macOS, Linux)
- Планшеты
- Игровые консоли
- Smart TV

### Максимальное количество устройств:
- До 20 одновременных подключений

## Устранение проблем

### Не могу подключиться к TravelPi
- Проверьте, что Raspberry Pi включен
- Убедитесь, что вы вводите правильный пароль
- Попробуйте перезагрузить роутер

### Нет интернета
- Проверьте подключение USB адаптеров
- Убедитесь, что выбран правильный режим
- Попробуйте переключить режим на "Авто"

### Медленный интернет
- Проверьте температуру процессора (не должна превышать 70°C)
- Уменьшите количество подключенных устройств
- Проверьте качество сигнала источника интернета

### Веб-интерфейс не открывается
- Попробуйте http://192.168.4.1 вместо travelpi.local
- Очистите кеш браузера
- Попробуйте другой браузер

## Безопасность

### Рекомендации:
1. ✅ Смените пароль администратора сразу после установки
2. ✅ Используйте сложный пароль для Wi-Fi (минимум 12 символов)
3. ✅ Регулярно обновляйте систему
4. ✅ Проверяйте список подключенных устройств
5. ✅ Блокируйте неизвестные устройства

### Обновление системы:
```bash
ssh admin@travelpi.local
sudo apt update && sudo apt upgrade -y
sudo reboot
```

## Полезные команды

### Проверка статуса:
```bash
# Статус точки доступа
sudo systemctl status hostapd

# Статус DHCP
sudo systemctl status dnsmasq

# Статус веб-сервера
sudo systemctl status nginx
```

### Просмотр логов:
```bash
# Логи подключений
tail -f /var/log/travelpi-connection.log

# Логи веб-сервера
tail -f /var/log/nginx/access.log
```
